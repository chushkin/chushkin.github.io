Execution plan

План обновления chushkin.github.io

  1. Собрать публичные данные Проверить карточки приложений, выбрать самые сильные продукты и ссылки на магазины.
  2. Сделать первый дизайн Собрать статический сайт с первым экраном, приложениями, магазинами и контактами.
  3. Добавить store links Вынести ссылки на Google Play, RuStore и App Store в карточки и общий блок магазинов.
  4. Подготовить публикацию Проверить адаптивность, SEO-мета, доступность и структуру файлов для GitHub Pages.
  5. Загрузить на GitHub Pages После подтверждения доступа опубликовать изменения в репозитории сайта.
